# Happy Cube
Happy Cube is a resource pack that makes the Sulfur Cube happy. 

# Compatibility
The pack is compatible from 26.2 Snapshot 1 to the most recent version.

# FAQ
**Q: Is there a sad version too?**  
A: Yes, search on modrinth ["Sad Cube"](https://modrinth.com/resourcepack/sad-cube).

# Incopatible mods/resource packs
No incopatible mods/resource packs reported (If you found one, you can make a issue on [github](https://github.com/b24fun/Happy-Cube/issues))
